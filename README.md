# Projet 1 : Cartographie et Vulnérabilisation (Équipe 3)

Ce dépôt contient la configuration et la documentation technique de la machine cible **SRVLX01** (Debian 13).

## 🚀 Objectifs du Sprint 1
* Isolation de la machine sur le réseau interne `intnet`.
* Configuration d'une IP statique (**172.16.10.6**).
* Installation de services vulnérables : **FTP (Port 21)** et **HTTP (Port 80)**.
* Audit de vérification via **Nmap** depuis la VM d'attaque.
